Get Marketing Measurements

Authorizations:
AuthBearer
query Parameters
products
Array of strings (Products)
campaigns
Array of strings
campaign_types
Array of strings
Items Enum: "always on" "branding" "performance"
channels
Array of strings
Items Enum: "tv" "radio" "ooh" "facebook" "print" "tiktok" "instagram" "db360 - prospecting" "db360 - retargeting" "google - branded search" "google - non-branded search" "dv360 - prospecting" "dv360 - retargeting" "folder"
start_date
string <date>
end_date
string <date>
offset
integer >= 0
Default: 0
page_size
integer ( 0 .. 100 ]
Default: 10
